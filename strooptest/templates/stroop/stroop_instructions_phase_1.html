Instructions for Phase 1

In this phase, you'll see a color word displayed in a specific color. Your task is to select the color of the word's text, regardless of the word's meaning. For example, if you see the word "BLUE" in red color, you should select "RED". Press the button that matches the color of the word's text as quickly as possible. Good luck!
